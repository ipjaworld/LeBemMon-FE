# 메이플랜드 레범몬

메이플스토리월드 메이플랜드 레벨 범위 몬스터 검색 서비스

## 기능

- 레벨 입력 시 해당 레벨 ±10 범위의 몬스터 검색
- 레벨 80 이상일 경우 70 이상 몬스터도 포함
- 곧 레범몬이 아니게 될 몬스터 표시

## 기술 스택

- Next.js 16.1.0
- React 19
- TypeScript
- Tailwind CSS 4

## 개발

```bash
# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# 프로덕션 서버 실행
npm start
```
